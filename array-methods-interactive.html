Why Array Methods Matter

❌ The Old Way (For Loop)
let numbers = [1, 2, 3, 4, 5];
let doubled = [];

for (let i = 0; i < numbers.length; i++) {
  doubled.push(numbers[i] * 2);
}
// [2, 4, 6, 8, 10]
✅ The Modern Way (Array Method)
let numbers = [1, 2, 3, 4, 5];
let doubled = numbers.map(num => num * 2);

// [2, 4, 6, 8, 10]
💡 Why Array Methods Win

More readable • Less code • Fewer bugs • Functional style • No manual indexing

Accumulating: reduce()

Σ
reduce()

Reduces an array to a SINGLE value by accumulating

Accumulation Flow (Sum Example):

Start: accumulator = 0
↓
Step 1: 0 + 1 = 1
↓
Step 2: 1 + 2 = 3
↓
Step 3: 3 + 3 = 6
↓
Step 4: 6 + 4 = 10
↓
Step 5: 10 + 5 = 15
↓
Final Result: 15 
let numbers = [1, 2, 3, 4, 5];
let sum = numbers.reduce((total, num) => total + num, 0);

console.log(sum); // 15
🧊 Snowball Analogy

Think of reduce() like a snowball rolling downhill. It starts small (initial value) and picks up more snow (each element) until you have one big snowball (final result)!

Finding Elements: find() & findIndex()

🔍
find()

Returns the FIRST element that passes a test

Search Flow:

Array:
5
12
8
130
↓ find(num => num > 10)
❌ 5 (too small) → ✓ 12 (found!) → ⏹️ Stop searching

let numbers = [5, 12, 8, 130, 44];

console.log(firstLarge); // 12 (stops at first match!)
// Real-world example: Find user by ID
let users = [
  { id: 1, name: 'Alice' },
  { id: 2, name: 'Bob' },
  { id: 3, name: 'Charlie' }
];

let user = users.find(u => u.id === 2);
console.log(user); // { id: 2, name: 'Bob' }
💡 find() vs filter()

find(): Returns FIRST matching element (or undefined)
filter(): Returns ALL matching elements in a new array

splice() - Modifies

Reordering: sort() & reverse()

⬆️
sort()

Sorts array elements (MUTATES original array)

Sorting numbers:

↓ sort((a, b) => a - b)
1

// ⚠️ Default sort treats numbers as strings!
let nums = [40, 100, 1, 5, 25];
nums.sort();
console.log(nums); // [1, 100, 25, 40, 5] - WRONG!

// ✅ Correct way: use compare function
nums.sort((a, b) => a - b);
console.log(nums); // [1, 5, 25, 40, 100] - CORRECT!
// Sort strings (works by default)
let fruits = ['banana', 'apple', 'orange'];
fruits.sort();
console.log(fruits); // ['apple', 'banana', 'orange']
⚠️ Important!

sort() MUTATES the original array. Always use a compare function for numbers: (a, b) => a - b for ascending, (a, b) => b - a for descending!

🌳 Which Method Should I Use?

What do you need to do?
↓
Add/Remove Items?
• End: push() / pop()
• Start: unshift() / shift()
• Middle: splice()
Transform Data?
• Same length: map()
• Filter items: filter()
• Single value: reduce()
Find Something?
• First match: find()
• Check existence: includes()
• Get index: indexOf()
Test Conditions?
• Any pass: some()
• All pass: every()
Reorder?
• Sort: sort()
• Reverse: reverse()
Extract/Combine?
• Extract: slice()
• Join: concat()
• String: join()
